Listv2

List can be used to display a list of pages. They can be defined either dynamically - by search query, tags or from a parent page - or as a static list of items.

Items with Link, Description and Date

The rendering of the list items can be configured to optionally display a link to the page, page description and last modified date.

  • Accordion 2019-01-22 Toggle panels of related content
  • Breadcrumb 2019-05-23 Display a breadcrumb navigation
  • Button 2019-01-09 Display a button or anchor button
  • Carousel 2019-02-15 Cycle through content panels
  • Container 2019-06-11 Group and layout components
  • Content Fragment 2019-01-16 Display a content fragment asset
  • Content Fragment List 2019-04-01 Display a list of content fragments
  • Download 2019-05-27 Display an asset for download
  • Embed 2019-08-29 Embed a third-party widget
  • Experience Fragment 2019-08-20 Display an experience fragment
  • Image 2019-02-15 Display an image asset
  • Language Navigation 2019-06-06 Display a language switcher
  • List 2019-01-16 Display a list of pages
  • Navigation 2019-11-11 Display a site navigation menu
  • Separator 2019-01-09 Display a section divider
  • Social Sharing 2019-01-09 Add social sharing links
  • Tabs 2019-01-22 Switchable content panels
  • Teaser 2019-03-05 Link an image and text
  • Text 2019-01-22 Display a rich text paragraph
  • Title 2019-01-16 Display a page heading
